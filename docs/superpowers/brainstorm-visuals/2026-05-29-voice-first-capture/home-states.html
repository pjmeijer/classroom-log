Gestures

Tap a tile → start recording inline (this view). Long-press a tile → opens the text-entry modal (existing screen with the mic button now wired up — voice works there too). Tap a Today's-note row → opens the text modal in edit mode. Tap Stop & save → Whisper transcribes; note saves; toast appears for 5 s. Tap Cancel → discard the recording, no note saved. Tap Undo (during toast) → deletes the just-saved note. Tap Edit (during toast) → opens the just-saved note in the text modal so you can fix a misheard word. Inside the text modal — tap the mic chip in the header → small recording bar slides in at the bottom of the modal; stop appends the transcript at the cursor, cancel discards it. Swipe-down on the text modal → same as tapping the ✕ (dismiss; prompts the discard sheet if dirty).
